# inicio.red.bo

Frontend PWA — Directorio digital para Bolivia y Latinoamérica.

## Stack
- HTML5 semántico + CSS3 Variables nativas + JavaScript Vanilla
- Cloudflare Pages (deploy automático)
- PWA con Service Worker + manifest.json
- Glassmorphism UI + Grid 5 columnas
- Bento Menu + Sidebar animado

## Estructura
```
inicio.red.bo/
├── index.html
├── manifest.json
├── sw.js
├── css/
│   ├── tokens.css
│   ├── base.css
│   ├── components.css
│   └── layout.css
└── js/
    ├── app.js
        ├── render.js
            └── sidebar.js
            ```

            ## Deploy
            Conectado a Cloudflare Pages → inicio.red.bo
